about

About
대화는 모든 일의 기본! 타인과의 대화를 중요시 여깁니다.

사이트를 구현함에 있어 다양한 사람들과의 협업이 있음을 알고 있습니다. 디자이너들과 협의를 하며 구현 가능한 것과 UI/UX 기반에서 사용자에게 적합한 레이아웃을 도출할 수 있습니다. 기업처의 수정 사항을 전달 받아 이를 기반으로 유지 보수를 이뤘으며 CS 업무를 맡은 적이 있습니다.

현실에 안주하지 않고 앞으로 나아가기 위해 노력합니다.

웹 퍼블리싱의 대한 이해도를 높이기 위해 노력합니다. JavaScript 기반인 gsap을 사용한 적이 있으며 swiper를 주로 사용하며 다양한 swiper api를 프로젝트를 진행하며 접할 수 있었습니다. 새로운 css 속성들을 알아보며 적용하며 코드를 최소한으로 구현하되 기능적으로 문제가 생기지 않도록 퍼블리싱하기 위해 노력합니다.

반응형은 기본이라 생각합니다.

기기별 해상도에 맞춰 디자인이 적절하고 사용자의 입장에서 불편함이 없도록 구현하기 위해 노력합니다. 1920px, 1280px, 1024px, 태블릿, 모바일로 분기점을 잡아서 작업을 하는 경우가 많았으며 확장 프로그램 중 모바일 시뮬레이터로 작업을 이어갔습니다.

지금까지 취득한 자격증은?

웹 디자인 기능사, 컴퓨터 그래픽스 운용 기능사, 컴퓨터 활용 능력 2급 자격증을 취득하였으며 ACP 자격증으로 포토샵, 일러스트레이터, 인디자인 분야를 취득했습니다. 후에 정보처리기능사를 취득할 계획을 하고 있습니다.

skill

Skill
HTML5
HTML5의 시멘틱 태그를 활용하여 Header, Main, Section, Footer 등의 요소로 구조화된 마크업을 구성할 수 있으며,
이를 통해 웹 문서의 의미론적 명확성과 접근성을 향상시킬 수 있습니다.
CSS3
기본적인 flex, grid를 사용해 구조를 clamp(), word-break, aspect-ratio, details, summary 등의
속성을 효과적으로 활용하여, 반응형·적응형 디자인 구현과 콘텐츠 가독성 향상에 기여할 수 있습니다.
JavaScript
JavaScript의 라이브러리인 Swiper.js 등의 라이브러리를 활용하여 반응형 슬라이더를 구현한 경험이 있으며,
이벤트 리스터, 클래스 토글, 탭 메뉴를 구현할 수 있습니다.
Figma
피그마를 사용해 디자인을 할 수 있습니다.
웹 페이지에 적절한 그리드를 입력할 수 있으며 플러그인을 사용할 수 있습니다.
Photoshop
이미지를 보정하거나 기본적인 포토샵 기능을 사용할 수 있습니다.
Illustrator
일러스트레이터를 기반으로 디자인 시안이 넘어왔기에
디자인 요소들을 적절하게 가공하고 기본적인 일러스레이터 사용이 가능합니다.
